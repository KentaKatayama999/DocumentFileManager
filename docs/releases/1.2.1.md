# DocumentFileManager 1.2.1 リリースノート

## 修正点

- UI スレッドから DocumentFileManager を初期化した際にデッドロックが発生することがあった問題を解消しました。
  - AppInitializer / DataSeeder / ChecklistLoader の非同期処理で ConfigureAwait(false) を徹底し、バックグラウンドスレッドで完結するように調整。
- 1.2.0 で導入したディレクトリ構成の単体テスト (PathSettingsTests) を継続利用できるようにプロジェクト設定を調整しました。

## 互換性

- API 互換性への変更はありません。
- 既存プロジェクトはパッケージを 1.2.1 に上げるだけで利用可能です。
